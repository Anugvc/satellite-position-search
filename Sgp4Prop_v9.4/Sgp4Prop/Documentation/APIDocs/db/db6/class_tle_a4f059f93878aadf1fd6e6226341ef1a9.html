◆ TleGPArrayToLines()

static native void TleGPArrayToLines ( double[]  xa_tle,
byte[]  xs_tle,
byte[]  line1,
byte[]  line2 
)
static

Constructs a TLE from GP data stored in the input parameters. This function only parses data from the input data but DOES NOT load/add the TLE to memory.
Returned line1 and line2 will be empty if the function fails to construct the lines as requested.
