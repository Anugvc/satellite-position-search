Sets the value of a numerical integration control parameter See SpGet4P for a list of the values for the xf_4P parameter. For flux, dca, and jpl file settings, the actual file loading happens when the first SpSatInit() is called. If the user needs to load these files instantly, please call the SpLoadFile() instead
