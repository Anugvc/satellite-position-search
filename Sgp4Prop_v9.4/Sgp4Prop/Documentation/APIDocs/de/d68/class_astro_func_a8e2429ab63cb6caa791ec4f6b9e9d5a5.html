◆ RAEToECI()

static native void RAEToECI ( double  theta,
double  astroLat,

static

Converts full state RAE (range, az, el, and their rates) to full state ECI (position and velocity) The xa_rae array has the following structure:

[3]: Range Dot (km/s) [4]: Azimuth Dot (deg/s) [5]: Elevation Dot (deg/s)

Note: This table may show index that are not used. Those are reserved for future use.

table
Name Index Description
XA_RAE_RANGE0Range (km)
XA_RAE_AZ1Azimuth (deg)
XA_RAE_EL2Elevation (deg)
XA_RAE_RANGEDOT3Range dot (km/s)
XA_RAE_AZDOT4Azimuth dot (deg/s)
XA_RAE_ELDOT5Elevation dot (deg/s)
XA_RAE_SIZE6
Parameters
thetaTheta - local sidereal time(rad). (in-Double)
astroLatAstronomical latitude (ded). (in-Double)
xa_raeAn array contains input data. see XA_RAE_? (in-Double[6])
senPosSensor position in ECI (km). (in-Double[3])
satPosSatellite position in ECI (km). (out-Double[3])
satVelSatellite velocity in ECI (km/s). (out-Double[3])
