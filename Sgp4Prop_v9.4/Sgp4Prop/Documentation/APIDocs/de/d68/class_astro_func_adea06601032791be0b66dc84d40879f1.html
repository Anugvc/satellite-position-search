◆ PosVelMuToEqnx()

static native void PosVelMuToEqnx ( double[]  pos,
double[]  vel,
double  mu,
double[]  xa_eqnx 
)
static

Converts position and velocity vectors to a set of equinoctial elements with the given mu value. This function is used when working with the SP propagator to get a more accurate set of equinoctial elements.

Note: This table may show index that are not used. Those are reserved for future use.
