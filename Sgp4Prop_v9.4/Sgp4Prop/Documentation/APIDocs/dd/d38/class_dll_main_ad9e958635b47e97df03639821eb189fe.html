static native int SetDupKeyMode ( int  dupKeyMode)
static

Sets DUPLICATION key mode - change the default behavior of returning a key which already exists in memory: zero versus actual value

Note: This table may show index that are not used. Those are reserved for future use.

table
Name Index Description
DUPKEY_ZERO0Returning (satellite/sensor/obs) key is zero to signify the existing data/key was already in memory
DUPKEY_ACTUAL1Return actual (satellite/sensor/obs) key regardless of the key/data duplication
Parameters
dupKeyModeDesired duplication key mode (see DUPKEY_? for available modes) (in-Integer)
Returns
0 if the set is successful, non-0 if there is an error.
