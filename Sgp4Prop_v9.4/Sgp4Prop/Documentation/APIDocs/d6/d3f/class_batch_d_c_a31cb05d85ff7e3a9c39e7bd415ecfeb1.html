static

Performs batch-least-square differential corrections to the specified satellite (VCM/SPVEC) and returns the corrected elements SGP4/SGP4-XP in form of a TLE Note: No need to initialize VCM before calling this method

Note: This table may show index that are not used. Those are reserved for future use.
