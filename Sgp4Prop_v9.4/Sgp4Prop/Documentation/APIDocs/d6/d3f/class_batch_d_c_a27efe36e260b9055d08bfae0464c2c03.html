Initializes DC parameters for the satellite using only obs that match the provided obs selection criteria (not all loaded obs) (Thread-Safe) See BatchDCGetParams for the structure of the xai_dcElts and xar_dcElts arrays. Note: This can be accomplished by using ObsGetSelected() and BatchDCInitSatObsKeys() as well
